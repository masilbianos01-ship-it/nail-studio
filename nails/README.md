# Nail Studio — Διαχείριση Πελατών & Ιστορικού

Web εφαρμογή σε Flask + SQLite για διαχείριση πελατών κομμωτηρίου/νυχιών.

## Τι κάνει

- Λίστα όλων των πελατών με αναζήτηση (όνομα, τηλέφωνο, email)
- Καρτέλα πελάτη: στοιχεία επικοινωνίας, αλλεργίες/ευαισθησίες, προτιμήσεις
- Ιστορικό επισκέψεων ανά πελάτη: ημερομηνία, υπηρεσία, χρώμα/σχέδιο, τιμή, σημειώσεις
- Προσθήκη / Επεξεργασία / Διαγραφή πελατών και επισκέψεων

## Εγκατάσταση & Εκτέλεση

```bash
pip install flask
python3 app.py
```

Μετά άνοιξε στον browser: **http://localhost:5000**

Η βάση δεδομένων (`salon.db`) δημιουργείται αυτόματα στον ίδιο φάκελο με το `app.py`
την πρώτη φορά που τρέχει η εφαρμογή.

## Δομή Project

```
nail_salon/
├── app.py                  # Flask routes + λογική βάσης
├── salon.db                 # SQLite βάση (δημιουργείται αυτόματα)
├── templates/
│   ├── base.html             # Κοινό layout
│   ├── index.html            # Λίστα πελατών / αναζήτηση
│   ├── client_form.html      # Φόρμα προσθήκης/επεξεργασίας πελάτη
│   ├── client_detail.html    # Προφίλ πελάτη + ιστορικό επισκέψεων
│   └── visit_form.html       # Φόρμα προσθήκης/επεξεργασίας επίσκεψης
└── static/
    └── style.css              # Στυλ (ροζ/nude θέμα)
```

## Πιθανές επεκτάσεις

- Σύστημα ραντεβού με ημερολόγιο
- Φωτογραφίες πριν/μετά ανά επίσκεψη
- Υπενθυμίσεις SMS/email
- Στατιστικά εσόδων ανά μήνα
- Login/auth αν θα το χρησιμοποιούν πολλοί υπάλληλοι
